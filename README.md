# Quote of the Day App Developed by Tadiwa Edrick Tembo

Quote of the Day is a modern and inspiring Android application designed to provide your daily dose of motivation. 

Featuring a clean Material 3 design, the app fetches random, thought-provoking quotes from the ZenQuotes API at the touch of a button.
# Key Features:
•
Daily Inspiration: 

Instantly fetch and display random quotes from world-renowned authors.
•
Favorites System: 

Save your favorite quotes to a local library using Room Database so you never lose an inspiring thought.
•
Seamless Sharing: 

Use the built-in Android share feature to send quotes directly to friends, family, or social media.
•
Modern UI/UX: 

Enjoy a sleek, professional interface built with MVVM architecture and Google's latest Material 3 design principles.
Stay motivated and share the wisdom with Quote of the Day.
